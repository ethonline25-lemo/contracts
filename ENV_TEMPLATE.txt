# □ Deploy LighthouseReceipt.sol to Calibration
# □ Deploy FVM Shop contract to Calibration (from GitHub)
#
# STEP 2: FUND CONTRACTS
# □ Approve LEMO tokens for TrustlessAgentFeedback contract
# □ Call fundContract() on TrustlessAgentFeedback
# □ Verify contract balance: call getLEMOBalance()
#
# STEP 3: UPDATE THIS FILE
# □ Replace all deployment addresses above
# □ Add your Lighthouse API key
# □ Save as .env file
#
# STEP 4: UPDATE EXTENSION
# □ Copy addresses to LEMO-extension/src/utils/contractConfig.js
# □ Run: npm run build (in LEMO-extension folder)
# □ Reload extension in browser
#
# STEP 5: TEST
# □ Connect wallet in extension
# □ Click "Buy Now" on a product
# □ Verify receipt is recorded on-chain
# □ Submit feedback
# □ Verify LEMO reward is received
# □ Check LEMO balance in wallet section
#
